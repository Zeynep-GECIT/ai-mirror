# AI Mirror

AI Mirror analyzes how vulnerable your career is to automation using Oxford University's Future of Employment research data and Google Gemini AI.

## Features
- Automation risk score for 702 occupations
- AI-powered career analysis
- Skills to develop to stay relevant
- 5-year career outlook

## Technologies
- Python
- Streamlit
- Google Gemini API
- Oxford Future of Employment Dataset
- TheFuzz (fuzzy matching)

## How to Run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Create a .env file and add your Gemini API key: GEMINI_API_KEY=your_key_here
4. Run: streamlit run app.py

## Data Source
Frey, C.B. and Osborne, M.A. (2017). The Future of Employment.
Oxford Martin School, University of Oxford.

## Disclaimer
This tool provides estimates based on academic research.
Results are not guaranteed predictions of your career future.
